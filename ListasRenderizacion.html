-Utilizando v-for para recorrer los elementos de mi dato asignado en la directiva de VueJS

-el, sirve para seleccionar al elemento padre

-data, sirve para definir nuestros datos

-methods, sirve para definir nuestros metodos que se ejucataran una vez realizada alguna accion de nuesta Wep App

v-model, sirve para definir el "value" por asi decirlo de ese elemento

v-on:keyup.enter, sirve como un evento, llama a un metodo definido en nuestra clase new Vue y empieza agregar a la lista ul(products) dinamicamente

v-text, sirve para mostrar un elemento de alguna lista luego de haber recorrido un v-for
